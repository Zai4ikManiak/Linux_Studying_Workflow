> [!NOTE]
> A Cheat Sheet for cofiguration of Monitors' parameters:

---

### Basic Syntax and Rules

Default Format:
```text
monitor = <name_of_the_screen>, <resolution>, <position>, <scale>
```


Monitorv2 Format:
```text
monitorv2 {

	output = <name_of_the_screen>
	mode = <resolution>
	position = <position>
	scale = <scale>
}
```
