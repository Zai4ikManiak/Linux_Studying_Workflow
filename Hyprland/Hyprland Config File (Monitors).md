### monitorv2 Function Parameters:

```text
monitorv2 {

	output = 
	mode = 
	position =
	scale = 
	transform = 
}
```
---
<details>

<summary><ins>output = </ins></summary>

>
> **Blank Field**:
> > Define a fallback rule to use when no other rules match
> 
> **Name of the Monitor**:
> > Set the rule of the monitor.

---
</details>
